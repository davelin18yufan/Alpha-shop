# Alpha Shop

Assignment of Alpha Camp Front-end course



## Getting Start

- Control input steps
- Increase and decrease cart amount
- press "確認下單", log user input and total price

## How to deploy

Command Line

    git clone https://github.com/davelin18yufan/Alpha-shop.git

Install package

    npm install

Run website

    npm start
